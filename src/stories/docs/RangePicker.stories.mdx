<br />

- Theme, Custom compoenents, Custom Days
- Support Jalali and Gregorian
- Use `day.js` It's 2KB immutable date

### Auto Responsive

By activate it, If your window growth from x-axis, the number of months will be increase and vice versa.

<RangePicker autoResponsive onChange={dates => console.log("dates", dates)} />

```jsx
<RangePicker autoResponsive />
```

### Custom Components

In this prop you have access to most of part of DatePicker and you can change every peace of it.

- Titles of weeks
- Icons in the header
- Wrapper of header of weeks
- Items of week's header
- Days

<RangePicker
  components={{
    days: ({ day, jalali }) => (
      <div style={{ fontSize: 12, color: "red" }}>
        *{dayjs(day).format("D")}*
      </div>
    ),
    header: {
      format: "YY**MMMM",
      monthIcons: {
        right: <span>next month</span>,
        left: <span>prev month</span>,
      },
      yearIcons: {
        right: <span>next year</span>,
        left: <span>prev year</span>,
      },
    },
    titleOfWeek: {
      titles: ["A", "B", "C", "D", "E", "F", "G"],
    },
  }}
  onChange={dates => console.log("dates", dates)}
/>

```jsx
<RangePicker
  components={{
    days: ({ day, jalali }) => (
      <div style={{ fontSize: 12, color: "red" }}>
        *{dayjs(day).format("D")}*
      </div>
    ),
    header: {
      format: "YY**MMMM",
      monthIcons: {
        right: <span>next month</span>,
        left: <span>prev month</span>,
      },
      yearIcons: {
        right: <span>next year</span>,
        left: <span>prev year</span>,
      },
    },
    titleOfWeek: {
      titles: ["A", "B", "C", "D", "E", "F", "G"],
    },
  }}
/>
```

### Disabled

By Disable the calendar you can't be able to select date it.

<RangePicker
  selectedDays={{
    from: dayjs().subtract(-2, "day").format("YYYY-MM-DD"),
    to: dayjs().subtract(-20, "day").format("YYYY-MM-DD"),
  }}
  disabled
  onChange={dates => console.log("dates", dates)}
/>

```jsx
<RangePicker disabled />
```

### Disabled Days

Disabled days.

<RangePicker
  disabledDays={[
    dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
  ]}
  onChange={dates => console.log("dates", dates)}
/>

```jsx
let disabledDays = [
  dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
];

<RangePicker
  disabledDays={disabledDays}
  onChange={dates => console.log("dates", dates)}
/>;
```
