<br />

#### Everything you want, you can find in React Trip Date. It's a fully customizable component that you can change to match with your design system.

<br />

- RangePicker, DatePicker, Multiple Month, Define number of selectable days
- Theme, Custom compoenents, Custom Days
- Support Jalali and Gregorian
- Use `day.js` It's 2KB immutable date

### Auto Responsive

By activate it, If your window growth from x-axis, the number of months will be increase and vice versa.

<DatePicker
  autoResponsive={false}
  numberOfMonths={1}
  onChange={dates => console.log("dates", dates)}
  dialog
/>

```jsx
<DatePicker autoResponsive numberOfMonths={1} />
```

### Custom Components

In this prop you have access to most of part of DatePicker and you can change every peace of it.

- Titles of weeks
- Icons in the header
- Wrapper of header of weeks
- Items of week's header
- Days

<DatePicker
  components={{
    days: ({ day, jalali }) => (
      <div style={{ fontSize: 12, color: "red" }}>
        *{dayjs(day).format("D")}*
      </div>
    ),
    header: {
      format: "YY**MMMM",
      monthIcons: {
        right: <span>next month</span>,
        left: <span>prev month</span>,
      },
      yearIcons: {
        right: <span>next year</span>,
        left: <span>prev year</span>,
      },
    },
    titleOfWeek: {
      titles: ["A", "B", "C", "D", "E", "F", "G"],
    },
  }}
  onChange={dates => console.log("dates", dates)}
/>

```jsx
<DatePicker
  components={{
    days: ({ day, jalali }) => (
      <div style={{ fontSize: 12, color: "red" }}>
        *{dayjs(day).format("D")}*
      </div>
    ),
    header: {
      format: "YY**MMMM",
      monthIcons: {
        right: <span>next month</span>,
        left: <span>prev month</span>,
      },
      yearIcons: {
        right: <span>next year</span>,
        left: <span>prev year</span>,
      },
    },
    titleOfWeek: {
      titles: ["A", "B", "C", "D", "E", "F", "G"],
    },
  }}
/>
```

### Disabled

### Disabled Days

Disabled days.

<DatePicker
  disabledDays={[
    dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
  ]}
  onChange={dates => console.log("dates", dates)}
/>

```jsx
let disabledDays = [
  dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
];

<DatePicker disabledDays={disabledDays} />;
```

### Selected Days

Initial Days that you can pass to DatePicker

<DatePicker
  selectedDays={[
    dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
    dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
  ]}
  onChange={dates => console.log("dates", dates)}
/>

```jsx
const selectedDays = [
  dayjs().subtract(-3, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-9, "day").format("YYYY-MM-DD"),
  dayjs().subtract(-12, "day").format("YYYY-MM-DD"),
];
<DatePicker selectedDays={selectedDays} />;
```
